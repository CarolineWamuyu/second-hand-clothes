# TukuStyle – Affordable Kenyan Fashion

**TukuStyle** is a modern, responsive website showcasing affordable second-hand (mitumba) fashion in Nairobi, Kenya. The site highlights weekly new stock, featured products, categories, testimonials, and company information in a clean, engaging layout.

## Project Overview

TukuStyle is designed for both desktop and mobile users, providing easy navigation through its responsive design. The website emphasizes:

- Affordable, high-quality mitumba clothing for men, women, and kids  
- Featured categories and hand-picked products  
- Customer reviews and testimonials  
- Contact and social media engagement  
- Calls to action for shopping and browsing categories  

---

## Features

- **Responsive Design:** Works on mobile, tablet, and desktop screens  
- **Navigation:** Desktop menu and mobile hamburger menu  
- **Hero Section:** Eye-catching banner with weekly stock highlights  
- **Product Categories:** Men, Women, Kids, and Accessories  
- **Featured Products:** Hand-picked items with pricing and size details  
- **Testimonials:** Customer reviews to build trust  
- **CTA Banner:** Highlights weekly new arrivals  
- **Footer:** Quick links, categories, contact info, and social media  
- **Floating WhatsApp Button:** Easy direct messaging for customers  

---

## Technologies Used

- **HTML5** – Semantic structure and accessibility  
- **CSS3** – Styling and layout (`style.css`)  
- **JavaScript** – Interactive elements and mobile menu (`script.js`)  
- **Google Fonts** – `Syne` and `DM Sans` for modern typography  
- **SVG Icons** – Logo and inline vector graphics
